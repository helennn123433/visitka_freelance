# Сайт команды

## Технологии

- **Vue 3** (Composition API + TypeScript)
- **Pinia** - Управление состоянием
- **Vue Router 4** - Навигация
- **Axios** - HTTP клиент
- **SCSS** - Стилизация
- **Jest** - Тестирование

## Установка

### Предварительные требования

- Node.js 16+
- npm 8+

### Установка зависимостей

```bash
npm install --legacy-peer-deps
```

> **Примечание:** Флаг `--legacy-peer-deps` необходим для совместимости TypeScript 5.3 с ts-jest 27

### Настройка окружения

Скопировать `.env.example` в `.env` и настроить переменные окружения:

```bash
cp .env.example .env
```

## Запуск

### Режим разработки

```bash
npm run serve
```

### Запуск тестового сервера (JSON Server)

```bash
npm run test-server
```

JSON Server запустится на порту 8080 с тестовыми данными.

### Сборка для production

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

### Запуск тестов

```bash
npm run test:unit
```

### Запуск линтера

```bash
npm run lint
```

## Архитектура проекта

Проект построен по методологии Feature-Sliced Design (FSD):

```
src/
├── app/              # Инициализация приложения, роутинг, глобальные стили
├── pages/            # Страницы приложения (композиция виджетов)
├── widgets/          # Композитные блоки страниц
├── features/         # Функциональные возможности (CRUD операции)
├── entities/         # Бизнес-сущности (services, examples, contacts)
└── shared/           # Переиспользуемый код
    ├── api/          # API клиенты
    ├── config/       # Конфигурация и константы
    ├── lib/          # Утилиты и composables
    ├── types/        # Общие типы
    └── ui/           # UI компоненты
```
